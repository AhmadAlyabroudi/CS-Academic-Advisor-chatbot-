# CS-Academic-Advisor-chatbot-
Graduation project for Computer Science students at Jordan University of Science and Technology.

## Main Features
- AI academic advisor chatbot
- GPA calculator
- Personalized and standard roadmap
- Virtual study rooms
- Admin panel

## Tech Stack
- Frontend: html/css/javascript
- Backend: Flask
- Database: MySQL
- Authentication: Firebase
